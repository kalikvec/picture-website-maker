Features include:

  • Auto creation of thumbnail files for all supported media types
  • Drag 'n Drop movement of thumbnails
  • Rename/Delete of files
  • Viewing/playback of media files - click on a thumbnail and it will open.
    • If it's an jpg, it'll open in the appliction in a window.  Click the image in the window to close.
    • If it's something other than a jpg image, it'll open in your computer's default application for that file type. For now this only works on Windows.
  • EXIF info displayed where available
  • Autosave of all metadata (the text you're typing to "cusomize" the thumbnail

Features still to come:

  • Scroll window for directories with lots of subdirectories
  • Keyboard shortcuts (ctrl-x, ctrl-v, etc)
  • Support for GIF, TIFF, and PNG formats (currently can't make thumbnails for them)
