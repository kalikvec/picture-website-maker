Nurflugel PictureBrowser

What exactly is Nurflugel Picture Browser?

Nurflugel Picture Browser is a program I've written to make generating web pages for your image/mp3/etc collections easy, quick, and painless.

I have a digital camera, and it's really easy to generate tons of pictures.  Making web pages out of that is a lot harder.  I've used Adobe Photoshop in the past to generate web pages for my pictures, but it lacks the ability to create captions easily.  If I went and commented the generated HTML directly, then any changes to the Photoshop collection would overwrite them.

In addtion, I wanted to be able to generate web pages for all media types - movies, mp3, avi, etc. Photoshop doesn't handle these at all.
